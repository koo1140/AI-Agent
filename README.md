# AI Architect Agent
**Your AI-Powered Architectural Assistant**
![](https://raw.githubusercontent.com/koo1140/AI-Agent/refs/heads/main/image-2.png)

A **Level 3 AI Agent** specializing in architectural design, technical documentation, automation, and research. This agent has **direct access** to tools for file manipulation, scripting, web research, and architectural workflows.

## ðŸ”§ **Tools & Capabilities**

### **1. File Operations**
- **Read/Edit Files**: Modify or analyze text-based files (e.g., `.py`, `.txt`, `.csv`, `.json`, `.md`, `.rvt` exports, `.dwg` data, etc.).
- **File Info**: Check file size, line count, or structure.
- **Create New Files**: Generate scripts, documentation, or configuration files.
- **Directory Exploration**: List and navigate project folders (`tree` command).

---

### **2. Architectural & BIM Tools**
- **Revit/Dynamo Automation**: Write, debug, or optimize Dynamo scripts for Revit.
- **Grasshopper/Rhino**: Generate or refine parametric scripts for Rhino.
- **CAD/BIM File Analysis**: Extract data from exports (e.g., schedules, layers, or metadata).
- **Construction Documents**: Review or generate technical drawings, specifications, or details.

---

### **3. Scripting & Automation**
- **Python Scripting**: Write or debug Python scripts for automation, data processing, or generative design.
- **Shell Commands**: Execute system-level tasks (e.g., file conversion, batch processing).
- **Parametric Design**: Create or refine Grasshopper/Dynamo definitions.

---

### **4. Research & Analysis**
- **Web Search**: Research building codes, materials, products, or architectural trends.
- **Site Analysis**: Analyze solar paths, wind patterns, or topography (via external data integration).
- **Code Compliance**: Check projects against local building codes or zoning laws.

---

### **5. Collaboration & Workflow**
- **Sub-Agent Delegation**: Spawn specialized sub-agents (e.g., "Revit Specialist," "Sustainability Consultant").
- **Memory Store**: Save and recall project-specific facts, standards, or preferences.
- **Project Documentation**: Generate `README.md`, reports, or summaries.

---

### **6. Supported File Formats**
- **Text-Based**: `.py`, `.txt`, `.csv`, `.json`, `.md`, `.xml`, `.html`, `.gh` (Grasshopper), `.dyn` (Dynamo).
- **CAD/BIM Exports**: `.ifc`, `.dxf`, `.svg`, `.obj` (metadata only).
- **Configuration**: `.ini`, `.yaml`, `.toml`.
